# ZUtil

Java 工具类，包括但不限于时间、正则、字符串、数字、集合、数组、Bean、随机数、加解密。

QQ 交流群：[273743748](https://jq.qq.com/?_wv=1027&k=yZfCzQ8f)，微信群加入 Q 群后艾特群主。

工具类使用请查看 [test](src/test/java/top/zhogjianhao)。

请使用最新版，过时方法只保留一个版本，删除时会在 READMEME 中说明。
