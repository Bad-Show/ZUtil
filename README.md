# ZUtil

Java 工具类，包括但不限于时间、正则、字符串、数字、集合、Bean、随机数。

工具类使用请查看 [test](src/test/java/top/zhogjianhao)。

请使用最新版，过时方法只保留一个版本，删除时会在 READMEME 中说明。
